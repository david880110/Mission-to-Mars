# Mission-to-Mars
Build a web application that scrapes various websites for data related to the Mission to Mars and displays the information in a single HTML page which includes:

* [NASA Mars News](https://mars.nasa.gov/news/?page=0&per_page=40&order=publish_date+desc%2Ccreated_at+desc&search=&category=19%2C165%2C184%2C204&blank_scope=Latest)
* [JPL's Featured Space Image](https://www.jpl.nasa.gov/spaceimages/?search=&category=Mars)
* [Mars Weather from twitter](https://twitter.com/marswxreport?lang=en)
* [Mars Facts](https://space-facts.com/mars/)
